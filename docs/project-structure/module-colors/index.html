Module - Colors

Conceptâ€‹

All colors are available in the special module Mui.Colors.

All colors are available as their corresponding name from the MUI Color Palette. They are typed as records and you can access the shades by the following rules:

  • Numeric keys are prefixed with a c
  • Keys that are prefixed by a char are lowercased

Let's take the color red as an example. The following table shows the equivalent keys for accessing its shades:
